--- layout: post title: skype на windows xp не хоче працювати? date: '2017-09-03T01:38:00.002+03:00' author: tivasyk tags: - комп'ютери - windows - підказки modified_time: '2017-09-03T22:16:48.954+03:00' blogger_id: tag:blogger.com,1999:blog-1498862426470895405.post-3658971674160074989 blogger_orig_url: http://www.tivasyk.info/2017/09/skype-windows-xp.html --- нині довелося мені трохи побуцатися з такою проблемою: skype на комп'ютері з windows xp відмовлявся конектитися, нарікаючи буцім на відсутність підключення до інтернету.

зрозуміло, що і підключення до інтернету в цей час було в порядку, і порти для skype у файрволі були відкриті, і логін/пароль безліч разів перевірені…

що ще пробував:
  • повністю відключав файрвол;
  • переконувався, що веб-версія skype працює з цим логіном/паролем;
  • видаляв і перевстановлював найсвіжішу версію skype для windows xp (7.36) з повним видаленням користувацьких налаштувань;
  • встановлював старшу версію skype (7.35);
  • переконувався, що internet explorer нормально «ходить» в інтернет (так, microsoft умудрилися і skype якимось чином прив'язати до двигунця ie, щоби ви й не сумнівалися);
  • поновлював сертифікати безпеки за допомогою rootupd.exe з сайту microsoft…
як я зрозумів, читаючи форуми, користувачів з цією проблемою безліч, і проблема таки виникла на стороні microsoft через чергове поновлення сервісу.

а ось що допомогло…

на комп'ютері з будь-якою старшою версією windows (в мене спрацювало на десятці, але потрібно взяти НЕ версію з новим інтерфейсом, яка йде в комплекті windows 10, а завантажити нормальний skype):
  • запустити skype і залогінитися!
  • поки skype працює — файловим менеджером зайти до теки користувацьких налаштувань windows (%appdata%) і зробити копію (краще архів) теки skype !
  • перекинути цю копію/архів на комп'ютер з windows xp.
 на комп'ютері з windows xp:
  • закрити skype повністю;
  • файловим менеджером зайти до теки користувацьких налаштувань windows (%appdata%), видалити теку skype  і замінити її копією з іншого комп'ютера (див. вище)
  • запустити skype — він покаже вікно логіну замість помилки підключення до інтернету, — і залогінитися як зазвичай.
наостанок — традиційні промені проносу палкі побажання довгих літ на адресу microsoft: молодці!
